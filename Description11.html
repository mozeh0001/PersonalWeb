Overview

This game puts players in the role of a kid testing Mr. Wondertainment’s whimsical toys in his new toy shop, inspired by the SCP fandom. This game further takes its inspirations from Willy Wonka and the Chocolate Factory and Cruelty Squad.
Each toy is strange and imaginative, offering unusual mechanics and playful surprises that transform ordinary play into whimsical experiences. Collect all 5 toys!
It incorporates special hardware such as an IoT glove, allowing players to physically "hold" objects, using special hand gestures instead of a mouse, drifting from the feats of ordinary gameplay.

IoT Integration + Ideation

Post-Production

Game Intro + Tutorial Explaining IoT Controls


  • Integrated IoT-controlled toys using a glove equipped with pressure sensors and an accelerometer.
  • Added five finger sensors and one wrist sensor to capture gripping, squeezing, and tapping.
  • Used an accelerometer to track 3D hand motions such as tilting, swiping, and lifting.
  • Linked physical gestures to virtual object interactions for natural, intuitive control.
  • Made each toy feel uniquely responsive through seamless physical-to-digital connection.

Sticky Hand

First Prototype + Joints

Second Prototype + Smoothed Joints + Added Scripts / Code


The sticky-fingered toy swings and stretches based on player gestures, reacting to grip strength and movement. Using the IoT glove’s sensors, players will be able to control its elasticity and motion in real time, creating playful, physics-driven interactions in Unity.


Groucho Glasses

Physical Inspo.

Final Creation with Post Processing


The "Groucho Glasses" let players reveal hidden objects. By making a fist with the IoT glove, the player lifts the glasses, activating a second camera view that uncovers what’s otherwise invisible in the game world.


Bubbles

Bubbles Expanding via Shader Graph

Bubbles Capturing Objects via Script / Code

Bubbles Wrap with Blend Shapes (Corresponding Level Design for Level that had Box Flaps... Volume Up for Pop Sounds!)


The Bubble Wand creates shimmering bubbles that expand until they capture nearby objects, making them float away. I used Unity’s Shader Graph to give the bubbles a shiny, translucent look, while velocity-based motion simulates their drifting, weightless behavior. Players can swing the wand by making a fist with the IoT glove, sending out bubbles that rise, float, and carry objects into the air.
