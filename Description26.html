USC Marketplace Application

CSCI 201 Final Group Project


  • Built a full-stack web app simulating an online marketplace with buyer/seller accounts, real-time messaging, and ratings.
  • Implemented secure login/registration using Spring Security Crypto and password encryption.
  • Designed multi-threaded messaging with STOMP/WebSockets for smooth, real-time conversations.
  • Used MySQL + data structures (sets, heaps, queues, maps) for listings, ratings, and chats.
  • Collaborated in a 7-person team, leveraging GitHub for version control and agile-style sprint planning.
